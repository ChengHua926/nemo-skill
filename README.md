# Nemo

Search engine for MCP tools and agent skills. Search 790+ MCP server tools and 760+ agent skills in one place.

**[findnemo.dev](https://findnemo.dev)**

## Install

```bash
npx skills add ChengHua926/nemo-skill
```

## What it does

Nemo indexes MCP tools and agent skills into a single searchable catalog. Agents use it to discover and call tools on demand via simple HTTP API.

See [SKILL.md](./SKILL.md) for the full agent instructions.
